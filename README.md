# ![UChat Logo](app/static/images/favicon.ico) UChat | Custom Twitch Overlay

Project is currently deployed at: https://chat.unii.dev/

A highly customizable Twitch chat overlay for OBS and other streaming programs.

## Want your app’s badges or your bot added to the overlay?

Create a pull request.

## Chat Customization Settings

- Messages in **bold**
- Messages in UPPERCASE
- Custom chat font
- Chat font color
- Font & badge size
- Font stroke
- Message shadow
- Emote size
- Text fade out
- Display badges
- Display channel points, subs, announcements
- Display bots in chat
- Custom user blacklist
- Custom prefix blacklist
- Moderation action effect
- Mentions colored
- Display 7TV Paints
- Display 7TV Paint Shadows
