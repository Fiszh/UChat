## Chat Customization Settings

- Make messages **bold**
- Make messages UPPERCASE
- Custom chat font
- Chat text color
- Adjust font size
- Font stroke
- Message shadow
- Emote size
- Custom message fade out 
- Show/hide badges
- Show/hide channel points redeems, sub messages, announcements
- Show/hide bots in chat
- User blacklist
- Prefix blacklist
- Toggle moderation actions
- Color mentions
- Show 7TV Paints
- Show 7TV Paint Shadows
